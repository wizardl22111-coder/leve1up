# صور المنتجات / Product Images

هذا المجلد مخصص لصور المنتجات المحلية.

## كيفية إضافة صورة جديدة:

1. **احفظ الصورة** في هذا المجلد
2. **استخدم اسم واضح** مثل: `15-project-ideas.png` أو `digital-products-guide.png`
3. **قم بتحديث** ملف `data/products.json` لاستخدام المسار الجديد

## مثال:

```json
{
  "id": 1,
  "name": "15 فكرة مشروع رقمي مربح",
  "image": "/images/products/15-project-ideas.png"
}
```

## الصور المطلوبة حالياً:

- ✅ `15-project-ideas.png` - للمنتج رقم 1 (15 فكرة مشروع رقمي مربح)
- ⏳ صورة المنتج رقم 2
- ⏳ صورة المنتج رقم 3

## ملاحظات:

- الصور يجب أن تكون بنسبة 1:1 (مربعة) للحصول على أفضل عرض
- الحجم المثالي: 800x800 بكسل أو أعلى
- الصيغة المفضلة: PNG أو JPG
- حاول أن يكون حجم الملف أقل من 500KB لسرعة التحميل
